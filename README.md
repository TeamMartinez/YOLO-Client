# YOLO-Client
This is the client for the YOLO App.

## Installation 
1. Download Node (v 5+). This should come with `npm`.
1. Run `sudo npm install -g gulp
1. On the directory, run `npm install`
1. Serve with `gulp`. Default port is `3000`.

## Technologies
- Semantic-UI (CSS FW)
- React
- Redux
- npm (package manager)
-   gulp (task-runner)
